# SuperDocuments — Documents Life Domain (consumer UX)

Consumer window onto **One Documents Library** (`AfterEcosystemDocuments`).

## Owns (UX)
Identity docs · Passport · Vehicle documents · Insurance · Warranty ·
Invoices · Contracts · Certificates · OCR · AI summaries · Expiration alerts

## Does not own
A private vault fork. Enterprise org docs are projections of the same
platform documents fabric (ADR-016).

## Ecosystem
Every Life Domain contributes/consumes documents via the shared library.
Find · Garage · Travel · Home · Health · Kids link entities to docs by ID.

## Admission
✅ People manage identity and legal documents for years.
